Welcome to GAC Academics project!

What is this website about?

This website is made for students to rate their professors and plan their 4-year academics. If you are a Gustie and have some coding skills in hand, feel free to contribute to this website by cloning the repo here.

How do I use this website?

To view professor ratings, simply navigate to the "Profs Rating", there you can view the list of professors, click on "View Detail" to view the detail for each professor and make a new rating. If you find any professor missing, please let me know. To add a 4-year plan, navigate to the "Classes Planning" section, input your start and end date at Gustavus, then the website will populate the semesters based on the start to end date. From there you can start adding your planned class.


I made this website since I saw the data for RateMyProfessor is outdated and would like to make a new one just for Gustavus students. If you found any issue, please let me know.
